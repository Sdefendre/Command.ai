# Command

Command is an AI-powered education platform for veterans - a multi-page Next.js 15 application helping veterans navigate VA benefits, disability claims, and financial literacy.

## Commands

- `pnpm dev` - Start dev server (localhost:3000)
- `pnpm build` - Production build
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint
- `pnpm format` - Format code with Prettier

## Code Style

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Components**: Functional, typed interfaces
- **State**: URL-based ('nugs'), Server Components (RSC)

## Structure

- `app/`: Routes (e.g., `app/about/page.tsx`)
- `components/`: UI components
- `constants/`: Static data (site config, blog posts)
- `public/`: Static assets
